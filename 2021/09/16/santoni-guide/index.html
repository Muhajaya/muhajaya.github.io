GSI (Santoni itu A-Only, jadi saat download pilih yg A, bukab A/B)

Sebelumnya apa itu GSI?

Generic System Image adalah dimana penyedia ROM hanya membuat system, dan vendor serta boot nya kita cari sendiri. Ini memudahkan untuk mencoba ROM tapi bagi device yg belum di support oleh tim developer. GSI itu universal, tapi dibagi menjadi 2, yaitu A dan A/B.

Contoh analogi nya adalah GSI itu seperti DVD Windows, yg bisa di install di Laptop (vendor) merek Asus, lenovo, dsb. Sedangkan ROM biasa yg kita kenal seperti macOs yang hanya bisa dijalankan si Komputer milik Apple.

  1. Buka TWRP
  2. Wipe : system, data, cache, dalvik, vendor/cust
  3. Install Treble Enabler.zip (8.1 dan 9.0 berbeda, but you can try)
  4. Install GSI.img (pilih install image di kanan bawah twrp di menu install)
  5. Reboot recovery
  6. Optional - Install gapps, magisk
  7. Jika mau pake microG dr pada gapps, disarankan flash microG nya setelah boot ke homescreen

note

Dirty flash adalah cara flashing rom tanpa melakukan wipe. Cara ini tidak disarankan, tp boleh dilakukan jika melakukan update ota.
Cara flash ROM Android 9 sama seperti cara flash ROM Treble, sedangkan GSI (A-Only) Android 9 sama seperti flash GSI (Treble enabler nya berbeda dengan Android 8, jadi pastikan tidak salah)

Cara flash ROM, GSI, atau file zip lainnya lewat komputer (adb - fastboot)

Cara ini digunakan jika kalian malas untuk mengcopy file yg di download dr komputer ke HP atau OTG.

Yg dibutuhkan :

  • Komputer bersistem operasi Windows, Linux, atau MacOS
  • Sudah terpasang adb dan fastboot pada system variable (bisa dipanggil langsung lewat cmd atau terminal, - tutorial nyusul)
  • Kabel data yg berfungsi
  • TWRP
  • Driver jika belum terpasang (linux dan MacOS gak perlu ea)
  • HP terbaca di komputer

Langkah-langkah :

Untuk flash File zip lewat adb

  1. Hubungkan hp ke komputer
  2. Dari TWRP, silahkan cari menu adb sideload (jika ingin flash ROM, wipe dahulu sebelum ke sini)
  3. Dari komputer, buka terminal/cmd, lalu ketik adb sideload C:\lokasi\namafile.zip atau jika ingin lebih mudah, di cmd/terminal, ketik adb sideload(spasi) terus seret file nya dari explorer ke cmd/terminal. Lalu enter.
  4. Tunggu hingga selesai

Untuk flash img lewat fastboot

  1. Hubungkan hp ke komputer
  2. Reboot ke fastboot/bootloader
  3. Dari cmd/terminal ketik fastboot device untuk cek, terhubung atau tidak
  4. Lalu ketik fastboot flash %partisi% C:\lokasi\namafile.img (atau seret seperti cara adb diatas)
  5. Partisi diatas diganti dengan partisi yg kita tuju, misal recovery (twrp/recovery), boot (kernel), system (GSI), dll.

    Contoh : fastboot flash system system.img

Cara membedakam ROM non treble, treble, gsi

Baca thread atau postingam ROM baikÂ², jika sudah terlanjur terdownload, silahkan lihat isi dalam file.

  • Jika file nya berbentuk img atau xz, maka itu GSI.
  • Jika file berbentuk zip, didalamnya terdapat file bernama vendor, maka itu treble. Jika tidak ada, maka itu non-treble.

note tidak ada mana yg lebih bagus dari rom GSI, treble, dan non-treble.

Cara mengatasi stuck charge di 90, 92, dll

Bila mengalami stuck, kemungkinan besar karena kesalahan di battery stat. Cara nya tinggal hapus file di data/system/batterystat.bin (melalui rootex atau twrp, terserah).

Jika masih stuck, silahkan coba cara berikut : http://c.mi.com/thread-364493-1-0.html

Cara Restore ROM Hasil backupan TWRP

Gunakannlah TWRP yg sama agar tidak terjadi error yg tidak diinginkan.

Sangat disarankan jika anda membuat backup TWRP dahulu sebelumnya (apa aja backup).

  1. Extract Zip backup TWRP
  2. Pindahkan ke folder TWRP/BACKUPS/%serial_number%/Nama_folder_backup (folder TWRP bisa berbedaÂ², contoh PBRP, FOX, dll) . Jika anda pernah backup sebelumnya, maka hanya tinggal pindahkan saja file hasil extract ke folder tersebut.
  3. Lalu reboot recovery dan masuk ke TWRP lalu Restore
  4. Pilihlah partisi System, Data, Boot, dan Vendor (jika ada). Jika terjadi error saat restore vendor, maka restore vendor nya belakangan, atau reboot recovery lg lalu restore vendor saja. Jika masih error, remount partisi vendor (TWRP>Mount>vendor/cust)

Perbedaan ROM Treble dengan Non Treble

Perbedaan ROM Treble dengan non treble adalah dibagian vendor dan boot. Pada ROM Treble, terdapat partisi vendor yg bisa kita ganti dengan vendor lain. Atau jika kita ingin mencoba ROM dr HP Redmi note 4, kita bisa mencoba nya dengan projek treble ini (dengan sedikit tweaks tentunya)

Patch Spoof Sign dengan Nanodroid-patcher (6, 7.1.2, 8.1, 9)

Syarat ROM harus keadaan deodex terlebih dahulu, caranya buka service.jar di system/framework . jika ada file .dex di dalam service.jar maka deodex. Jika belum deodex, silahkan lihat no 14. Patch ini gunanya untuk yg menggunakan microG atau aplikasi yg bertujuan untuk menyamar sebagai aplikasi lain. Hati2 dalam menggunakan, jangan asal install sembarangan aplikasi!

  1. Download patcher di website nanolx
  2. Flash via TWRP atau Magisk (Ajaib nya bisa di flash di magisk, tp gajadi module, jangan kemana2 saat flash di magisk)
  3. Tunggu sekitar 10-15 menit
  4. Reboot system
  5. Sign spoof otomatis terpasang di Setting > app > permission

Alternatif Deodex dengan vdex2dex.com

Konvert berkas vdex ke dex. Kompatible dengan Android P.

Jika terdapat berkas /system/framework/oat/[arch]/services.vdex, dimana [arch] adalah arsitektur perangkat (arm, arm64, x86 or x86_64), bisa dengan menggunakan website ini dengan deodex.

Setelah jadi service.jar nya, tinggal push ke system/framework

Buka http://vdex2dex.com/
